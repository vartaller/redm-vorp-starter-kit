# varta_medic

NPC-доктор для VORP. Спавнит педа на каждой локации, игрок подходит и покупает лечение за деньги. Не требует игроков с джобом доктора.

## Текущие возможности
- Лечение здоровья
- Восстановление выносливости
- Полное лечение (здоровье + выносливость)
- Уникальная модель доктора на каждой локации

## Настройка (`config.lua`)

```lua
Config.Prices      -- стоимость каждой услуги
Config.HealValues  -- количество восстановления (0–100)
Config.Locations   -- координаты, модель, название для каждой локации
```

Для корректировки позиции NPC используй координаты из EasyAdmin (F10 → телепорт → скопируй coords).
